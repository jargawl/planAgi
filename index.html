Program 01.08.2026 – 01.11.2026 · 13 tygodni
Zasada bezpieczeństwa: przy bólu, mrowieniu lub drętwieniu w nogach — przerwij ćwiczenie. Ten plan nie zastępuje konsultacji z fizjoterapeutą; warto sprawdzić stan mięśni brzucha (rozejście) przed intensywną pracą nad core.
Jak to działa Poniżej masz gotowy kalendarz na cały program — każdy dzień od 01.08 do 01.11.2026 ma przypisany konkretny trening (albo dzień wolny). Żeby nie było "w kółko to samo":
  • Poniedziałek / Środa / Piątek — trening siłowy A, B lub C — kolejność rotuje co 3 tygodnie, więc np. Poniedziałek nie zawsze jest tym samym treningiem
  • Sobota — bieganie w terenie
  • Niedziela — Regeneracja (mobility + sauna), obowiązkowo
  • Wtorek / Czwartek — dni wolne
Konkretny dzień Ci nie pasuje? Możesz go swobodnie przesunąć — kalendarz to szkielet, nie sztywna wyrocznia.
